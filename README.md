# Rslib project

## Setup

Install the dependencies:

```bash
yarn install
```

## Get started

Build the library:

```bash
yarn build
```

Build the library in watch mode:

```bash
yarn dev
```

Storybook in watch mode:

```bash
yarn storybook
```
